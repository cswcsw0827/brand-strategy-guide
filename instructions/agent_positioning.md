## 3. 阶段二：内容生成

**核心原则**: Agent B 的战略定位不是"创始人想成为什么"，而是"生态位需要什么身份来占据"。brief.md 的创始人动机是原材料，Agent A 的生态位判断是过滤器，B 的输出是被过滤器加工后的成品。

### 必须遵守的输出规范
- **一致性铁律**: 模块 9 的定位方法 → 模块 10 的定位语 → 模块 12 的原型选择 → 模块 13 的品牌故事 → 模块 15 的视觉方向，必须形成一条可追溯的逻辑链
- **所有结论必须有推导过程**: 不只要"是什么"，更要"为什么是这个而不是那个"
- **案例分析**: 每个模块至少 1 个对标案例（可以是正例也可以是反例）

### 模块清单

#### 模块 9：品牌定位方法选择
- 回顾 Agent A 的生态位判断（目标人群 + 差异化空间 + 竞争空白）
- 从 4 种定位方法中选定 1 种：品类占位型 / 竞争对立型 / 情感价值型 / 价值主张型
- 详细推导过程：为什么选 A 而不选 B/C/D？
- 所选方法在该行业的适用性论证（附行业案例佐证）

#### 模块 10：品牌定位陈述
- 一句话定位语（满足 7 要素检查：品类锚定 / 人群锁定 / 差异承诺 / 信任状 / 情感共鸣 / 竞争区隔 / 可持续性）
- 定位语拆解：每个要素如何在定位语中体现
- 扩展定位 3 层：功能定位 → 情感定位 → 社会定位
- 与 Top 3 竞品定位的差异化图示（文字描述版）

#### 模块 11：品牌核心价值体系
- 使命、愿景、价值观（必须自洽：使命说的事 → 愿景要达成的状态 → 价值观是路上的行为准则）
- 品牌金字塔：底部（产品属性）→ 中部（功能利益）→ 上部（情感利益）→ 塔尖（品牌信念）
- 品牌价值主张画布：用户痛点 × 解决方案 × 差异化优势

#### 模块 12：品牌人格化（12 原型）
- 基于定位方法和生态位，从 12 原型中选择 1 个主原型 + 可选 1 个辅原型
- 原型选择推导：定位 → 原型三步筛选（写出每一步的逻辑）
- 原型落地指南：品牌语调（TOV）、视觉性格、行为模式、禁忌清单
- 该原型在行业中的优劣势分析

#### 模块 13：品牌故事
- 基于创始人真实动机（brief.md §6.4），按三要素结构创作：
  - 初心（My Heart）：那个"必须做这件事"的瞬间
  - 态度（My Attitude）：笨拙的坚持——别人怎么做，我们偏不
  - 承诺（My Promise）：把初心和态度绑定到用户利益
- 故事类型与定位方法的对应（英雄之旅 / 智者启蒙 / 凡人觉醒 / 关怀者守护等）
- 故事可传播性检查：能否被用户复述？能否在 3 分钟内讲完？

#### 模块 14：品牌命名与话语体系
- 场景判断：品牌名已有（走评估流程）还是待定（走创作流程）
- 如已有品牌名：6 维诊断报告（记忆度 / 传播度 / 联想度 / 区隔度 / 延展度 / 商标可行性）
- 如待定：5 步创作流程（定位提取 → 语义发散 → 音韵筛选 → 商标初筛 → 用户测试建议）
- 品牌话语体系：品类话术 + 品牌话术 + 产品话术 3 层架构

#### 模块 15：品牌视觉方向
- 第0层：心智锚定（三个必答题——定位、人群、情绪 → 视觉关键词）
- 第1层：视觉定位三角法（竞品空白 × 用户审美 × 品牌基因 → 视觉方向一句话）
- 第2层：四大符号方向（色彩符号 / 图形符号 / 字体符号 / 材质符号）——给出方向性建议，非最终设计稿
- 与 brief.md §6.5 创始人审美参照的关联说明（什么采纳了什么转化了什么拒绝了，为什么）

#### 模块 16：产品策略
- 产品线规划逻辑（引流品 / 利润品 / 形象品 / 防御品）
- 爆品策略：哪一款主打？为什么它具备爆品潜力？
- 产品与定位的一致性检查：产品是否在兑现定位承诺？
- 产品差异化：成分/技术/设计/体验/包装，哪个维度最有竞争力？

#### 模块 17：定价策略
- 定价方法选择（成本导向 / 竞争导向 / 价值导向 / 心理定价）
- 基于生态位的价格带建议（与 Agent A 竞品矩阵对应）
- 定价与定位的一致性：价格是否在"说"和定位一样的话？
- 价格弹性预估与调价空间
